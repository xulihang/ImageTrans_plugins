# oneocr

Windows 11 snipping tool's OCR.


For Windows 10:

Download Microsoft.ScreenSketch_2022.2601.2.0_neutral_~_8wekyb3d8bbwe.Msixbundle, extract the following files and put them under the oneocr folder in ImageTrans's directory.

* oneocr.dll
* oneocr.onemodel
* onnxruntime.dll

You can search `https://apps.microsoft.com/detail/9mz95kl8mr0l` in <https://store.rg-adguard.net/> to find the download link.
